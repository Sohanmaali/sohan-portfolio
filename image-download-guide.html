Project Images for Your Portfolio

How to Download and Use These Images

  1. For each project, choose one of the suggested images that best represents the project
  2. Right-click on the image and select "Save Image As..."
  3. Save the image to your /public/images/projects/ directory
  4. Name the file according to the project number (e.g., project1.jpg, project2.jpg, etc.)
  5. For detail images, save additional images with names like project1-detail1.jpg
